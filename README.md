# Afnuz Academy Website

This is a full-stack web application built using HTML, CSS, JavaScript, and Node.js.

## Features
- Responsive UI
- Teacher showcase section
- Contact form with backend integration
- Clean and user-friendly design

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express

## How to Run
1. Install dependencies:
   npm install

2. Start server:
   node server.js
